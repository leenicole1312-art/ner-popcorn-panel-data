# NER Popcorn Panel Data

Validated public data feed for **The NER Popcorn Panel**.

## Manual run

Open **Actions** → **Update IMDb Top 250** → **Run workflow**.

The workflow also runs weekly on Sunday morning. It refuses to publish unless
it finds exactly 250 unique films with valid IMDb IDs and ranks 1–250.

## Feed

https://raw.githubusercontent.com/leenicole1312-art/ner-popcorn-panel-data/main/data/top250.json
